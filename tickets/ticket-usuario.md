# Ticket: Usuario bloqueado

## Problema
El usuario no puede acceder al sistema.

## Diagnóstico

1. Validar credenciales
2. Revisar estado de cuenta
3. Confirmar bloqueo por intentos fallidos

## Solución

1. Desbloquear usuario
2. Resetear contraseña
3. Solicitar cambio inmediato

## Resultado
✅ Acceso restablecido
