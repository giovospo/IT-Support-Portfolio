# Ticket: Computador lento

## Problema
El usuario reporta lentitud general en el equipo.

## Diagnóstico paso a paso

1. Verificar uso de CPU y memoria
   Ctrl + Shift + Esc

2. Identificar procesos pesados
3. Revisar programas al inicio

4. Verificar espacio en disco

## Solución

1. Cerrar procesos innecesarios
2. Deshabilitar inicio automático
3. Liberar espacio en disco
4. Ejecutar limpieza:
   cleanmgr

## Resultado
✅ Mejor rendimiento del sistema
