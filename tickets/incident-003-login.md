# Incident INC-003 - User Account Locked

## 📌 Category
Access Management

## 👤 User
Corporate User

## ⚠️ Impact
High – Cannot access system

## 📝 Description
User account locked after multiple failed login attempts.

## 🔎 Troubleshooting Steps

1. Verified user account status in Active Directory
2. Confirmed account lock
3. Reset password

## ✅ Resolution
Account unlocked and password reset.

## 📊 Status
Resolved

## 🧠 Knowledge
Repeated failed attempts triggered security lock.
