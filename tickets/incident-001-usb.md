# Incident INC-001 - USB Device Not Recognized

## 📌 Category
Hardware / Peripherals

## 👤 User
End User – Finance Department

## ⚠️ Impact
Medium – User cannot transfer files

## 📝 Description
User reports that USB devices are not recognized by the system.

## 🔎 Troubleshooting Steps

1. Verified USB device works on another machine ✅
2. Checked Device Manager → USB controllers showed error ❌
3. Reinstalled USB drivers
4. Restarted system

## ✅ Resolution
USB drivers updated and reinstalled successfully.

## 📊 Status
Resolved

## 🧠 Knowledge
Issue caused by corrupted USB drivers.
