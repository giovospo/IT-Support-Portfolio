# Incident INC-002 - No Internet Connectivity

## 📌 Category
Network / Connectivity

## 👤 User
Remote Employee

## ⚠️ Impact
High – User cannot perform work tasks

## 📝 Description
User reports no internet access.

## 🔎 Troubleshooting Steps

1. Verified WiFi connection ✅
2. Executed:
   ipconfig
   ping 8.8.8.8 ❌
3. Renewed IP address
4. Verified DNS settings

## ✅ Resolution
Configured DNS manually (8.8.8.8)

## 📊 Status
Resolved

## 🧠 Knowledge
DNS misconfiguration caused connectivity issue.
